## Features
+ Blackout
+ Nearby user information
+ Terminal
+ Explosion
+ [Video Preview](https://streamable.com/sruub5)


## How to use
+ add item "hackerphone"
+ server.cfg ```ensure um-hackerphone```
+ use item {show nui}
+ hide nui key {backspace}
